SAE 1.05 : Traitement de la Data (Open Data)

Semestre 1 • 25h

Contexte : L'objectif était de valoriser des données publiques brutes (fichiers CSV volumineux) en les transformant en une application web graphique.

Architecture du projet :

  • Back-end (Python) : Développement de scripts pour nettoyer les données (gestion des valeurs manquantes), filtrer les colonnes pertinentes et convertir le format CSV en JSON structuré. Utilisation de structures de données (listes, dictionnaires).
  • Front-end (JS/PHP) : Intégration des données JSON dans une bibliothèque graphique (ex: Chart.js) pour visualiser les statistiques.

Défi technique :

Le tri des données a été complexe : les fichiers sources contenaient des millions de lignes avec des formats parfois incohérents. J'ai dû écrire des algorithmes de validation rigoureux avant la conversion.

SAE 2.1 : Automatisation & Scripting Hardware

Semestre 2

Contexte : Automatisation des tâches administratives dans le projet de pépinière d'entreprise.

Réalisation : Reconnaissance de Badges

J'ai développé un script Python complet capable de s'interfacer avec un lecteur de badges (RFID/NFC). Le programme :

  1. Écoute le port série/USB en attente d'un scan.
  2. Récupère l'ID unique du badge.
  3. Interroge une base de données locale ou un fichier CSV pour identifier l'utilisateur.
  4. Déclenche une action (ouverture de session, log d'accès).

Cela démontre ma capacité à lier la programmation logicielle avec des événements physiques (Hardware).
