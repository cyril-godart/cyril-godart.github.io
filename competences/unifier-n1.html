Compétence : Développer

Niveau 1 : Web Statique, Dynamique & Conteneurisation (S1 & S2)

SAE 1.04 : Conception Web & Identité Numérique

Semestre 1 • 10h

Contexte : Création de ce portfolio pour présenter mon profil professionnel.

Compétences mises en œuvre :

  • HTML5 / CSS3 : Structuration sémantique des pages et stylisation personnalisée (sans framework au départ pour bien comprendre les bases).
  • Responsive Design : Utilisation des Media Queries pour adapter l'affichage aux mobiles. (C'était la partie la plus difficile : gérer le repositionnement des éléments sans casser la mise en page).
  • Conformité : Rédaction des mentions légales, respect des droits d'auteur (crédits images/polices) et accessibilité de base.
Voir la version V1 (Wix/Static)

Projet S2 : Application Web Full-Stack & Docker

Semestre 2 • 25h

Contexte : Transformation du site statique en une véritable application dynamique, hébergée de manière professionnelle.

Technologies & Architecture :

Backend (Flask) : Création d'une API et de routes en Python. Gestion des sessions utilisateurs.
Base de Données (SQL) : Modélisation des données (Utilisateurs, Projets) et requêtes SQL sécurisées contre les injections.
DevOps (Docker) : Conteneurisation de l'application. Rédaction d'un docker-compose.yml pour lancer simultanément le serveur Web et la Base de Données dans des environnements isolés.

Bilan :

Ce projet m'a fait passer du statut d'intégrateur web à celui de développeur Full Stack junior, capable de déployer une application sécurisée.
